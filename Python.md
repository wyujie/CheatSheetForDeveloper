# Python

## 变量/数据结构
#### 定义
```
a = 1
b = 'B'

# List
c = []
c.append(1)
c += [2,3,4]

# Dict
d = {
  'k1': 'v1'
}
d['k2'] = 'v2'


#Set
e = set()
e.add(1)

# Tuple
f = (1, 2, 3)
```

#### 逻辑控制/迭代
###### Dict
```
if 'k1' in d:
  print(d['k1'])

for k, v in d.items():
  print(k, v)

```

###### List/Tuple/Set
```
for c1 in c:
  print(c1)
```
